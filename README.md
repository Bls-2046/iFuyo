# 辅佑 iFuyo
星辅天衢阔，泽佑百卉生。云移千嶂暗，露结一灯明。

# 辅助功能集合：

一、通过学号获取微门户信息；

二、设置日程安排到点自动在微信和桌面提醒事务；

三、DeepSeek 对话；

四、上传网页图标和 url 快速访问网页；

五、再说
